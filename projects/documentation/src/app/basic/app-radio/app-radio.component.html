Radio Group

A radio group is a type of UI control that allows users to select one option from a list of mutually exclusive options. A radio group is typically implemented using a set of radio buttons, each of which represents a single option. When a user selects one of the radio buttons, it becomes checked and the other buttons in the group become unchecked.

The radio group provides a way for users to make a single selection from a list of options, and the individual radio buttons within the group ensure that only one option can be selected at a time. This makes radio groups a useful choice for scenarios where the user needs to choose from a small, predefined set of options.
