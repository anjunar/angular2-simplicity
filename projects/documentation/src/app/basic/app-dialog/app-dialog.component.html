As Dialog

This is a dialog that is rendered into the viewport. The dialog can be started either with the action attribute, or generated with the windowManager. The only important thing is the modal flag in the view that represents the dialog. Width and Height can also be given, otherwise the size of the dialog is calculated from the content.
