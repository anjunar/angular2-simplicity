Dom Select

A native implementation of the select element. Via bind: model you can access the value in two-way mode be taken from the option element. Native select works well when there are only a few values about the options element there. If the values exceed a certain number of possibilities, then it is advisable to use the lazy select.

DOM-API

Element Attribute Description
select model The two-way model of the component
option value The value of the option element
