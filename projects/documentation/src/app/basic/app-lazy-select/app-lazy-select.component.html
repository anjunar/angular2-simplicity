Lazy Select

This is a lazy select that receives the values from a callback. There are only 5 items at a time displayed. The model consists of one of the selected records when the record was selected. In contrast, the model of the native select consists of the value of the option element. In addition to the lazy Select there is the as-input-container to turn the lazy Select into a Material Design Select.

DOM-API

Element Attribute Description
as-input-container placeholder The Placeholder text
dom-select items The items data source
dom-select model Two way data binding
dom-select multiselect True or false for multi selection

Lazy Multi Select

With the attribute "multiselect" the select can be converted into a multi-select. In the overlay there is an icon in the top left to indicate the selected items. Is that select in Multi-select mode then the model consists of an array with the selected elements.
