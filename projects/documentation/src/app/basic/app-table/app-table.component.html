Table

A paginated table is a type of UI control that is used to display a large amount of data in a tabular format. It is typically used when there is too much data to display on a single page, and the data needs to be divided into multiple pages for better readability and performance.

In a paginated table, the data is divided into pages, and the user can navigate between pages by clicking on buttons or links that represent the page numbers. The user can also adjust the number of items displayed per page, and some implementations also allow the user to sort and filter the data.
