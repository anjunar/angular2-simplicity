Tabs and Pages

Tabs are a type of UI control that allow users to switch between different content views within a single container. Tabs are usually represented by a row of clickable labels, each of which represents a different view or section of content. When a user clicks on a tab label, the corresponding content is displayed, while the content associated with the other tabs is hidden.

Tabs are commonly used in web and mobile applications to organize and present large amounts of content in a anageable way. They can be used to display different sections of a form, to switch between different views in an application, or to navigate between different sections of a website.
