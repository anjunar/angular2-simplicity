Dom Checkbox

Input elements of type checkbox are rendered by default as boxes that are checked (ticked) when activated, like you might see in an official government paper form. A checkbox allows you to select single values for submission in a form (or not).

A DOMString representing the value of the checkbox. This is not displayed on the client-side, but on the server this is the value given to the data submitted with the checkbox's name.

With the bind:model Attribute you can bind the value of the Checkbox to a property of your choice.

DOM-API

Element Attribute Description
as-checkbox-container placeholder The Placeholder text
input model The Bind variable that can be bound
